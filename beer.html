BeerDotCom

A simple online beer shop with a search bar and paginated results.

Technology used: React, HTML, CSS, and JavaScript (MaterialUI for some component designs)

BeerDotCom is a web app that retrieves beers according to the user search input.

  • fully responsive across multiple devices (phone, tablet, laptop, and desktop computers)
  • Makes API calls to PunkBeer API and handle loading, resolved and error conditions
  • Pagination
